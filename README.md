# Sistema de Sócio Torcedor - Ponte Preta

Projeto desenvolvido em Python para o PIM do curso de Análise e Desenvolvimento de Sistemas.

O sistema simula um gerenciamento de sócio torcedor da Ponte Preta, permitindo cadastro de usuários, login, compra de ingressos e acesso à carteirinha digital.

## Funcionalidades
- Cadastro de usuário
- Login
- Compra de ingressos
- Validação de entrada
- Sistema de sócios
- Carteirinha digital
- Salvamento de dados em JSON
  
## Tecnologias usadas
- Python
- JSON
  
## Como executar
1. Baixe os arquivos do projeto
2. Abra a pasta no VS Code
3. Execute o arquivo principal:
```text
  python main.py
```
